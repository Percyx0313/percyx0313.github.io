# MAiTH LAB 螺旋：從幾何到機率分布的奇妙旅程

在數學與機器學習的交會處，我們常常需要將幾何圖形轉換為機率分布。今天，讓我們用**MAiTH LAB**的logo——一條優雅的螺旋曲線——來深入理解這個轉換過程。

## 🌀 認識 MAiTH LAB 螺旋

### 📐 數學定義

我們的螺旋並不是隨意畫出的，而是有精確數學定義的參數化曲線：

```
螺旋的極座標定義：
θ(t) = π + t × (2.5 × 2π - π/4),  t ∈ [0, 1]

半徑成長函數（softplus 型態）：
r(t) = r₀ + scale × [log(1 + e^(k(t - δ)))] / [log(1 + e^(k(1 - δ)))]

其中：
- r₀ = 0.001 (初始半徑)
- scale = 1.5 (最大放大尺度)  
- k = 5 (曲線平滑程度)
- δ = 0.2 (開始放大的偏移點)
```

### 🎯 轉換為直角座標

```
x(t) = r(t) × cos(θ(t))
y(t) = r(t) × sin(θ(t))
```

這個函數描述了一條從中心點開始，逐漸向外擴展的螺旋曲線。

## 🎲 什麼是 Distribution Variable？

**Distribution Variable（分布變數）**是機率論中的核心概念，它描述了：

1. **隨機變數的可能取值**
2. **每個取值的發生機率**
3. **變數之間的關係**

### 🔍 從函數到分布的轉換

我們的螺旋曲線原本是一個**確定性函數**：給定參數 `t`，就能得到確定的 `(x, y)` 座標。

但要將它轉換為**機率分布**，我們需要引入**隨機性**：

```
資料生成過程：
1. t ~ Uniform(0, 1)  # 參數 t 從均勻分布抽樣
2. z = [x(t), y(t)] + ε  # 在螺旋點上加入隨機擾動
3. ε ~ N(0, σ²I)  # 高斯雜訊
```

## 📊 理解分布的結構

### 🧬 分布的「骨架」

螺旋曲線就像是分布的**骨架**：

- **主要結構**：數據主要集中在螺旋附近
- **局部變異**：每個點周圍有高斯分布的雜訊
- **全局模式**：整體呈現螺旋狀的機率密度

### 🎨 視覺化理解

想像你在螺旋路徑上撒下許多小球，每個小球會在落地時稍微彈跳偏移。最終的分布就是所有小球的位置集合。

## 🔧 Distribution Fitting 的核心思想

### 💡 什麼是 Distribution Fitting？

**Distribution Fitting**是機器學習中的重要技術，目標是：

1. **觀察數據**：我們有一堆散點數據
2. **推測結構**：猜測背後的生成過程
3. **參數估計**：找到最佳的參數值
4. **模型驗證**：檢驗模型的準確性

### 🎯 以螺旋為例

假設我們觀察到一堆呈螺旋狀分布的數據點，我們可能會：

1. **選擇模型**：假設數據來自「螺旋 + 高斯雜訊」
2. **估計參數**：找到最佳的 `r₀, scale, k, δ, σ`
3. **生成新數據**：用擬合的模型產生新的樣本點

## 🧪 實戰演練：生成螺旋分布

### 💻 Python 實作概念

```python
import numpy as np
import matplotlib.pyplot as plt

def maith_spiral(t, r0=0.001, scale=1.5, k=5, delta=0.2):
    """MAiTH LAB 螺旋生成函數"""
    # 角度
    theta = np.pi + t * (2.5 * 2 * np.pi - np.pi/4)
    
    # 半徑 (softplus 型態)
    r = r0 + scale * np.log(1 + np.exp(k * (t - delta))) / np.log(1 + np.exp(k * (1 - delta)))
    
    # 直角座標
    x = r * np.cos(theta)
    y = r * np.sin(theta)
    
    return x, y

def generate_spiral_distribution(n_samples=1000, noise_std=0.1):
    """生成螺旋分布的樣本"""
    # 隨機參數
    t = np.random.uniform(0, 1, n_samples)
    
    # 螺旋座標
    x, y = maith_spiral(t)
    
    # 加入高斯雜訊
    x += np.random.normal(0, noise_std, n_samples)
    y += np.random.normal(0, noise_std, n_samples)
    
    return np.column_stack([x, y])

# 生成樣本
samples = generate_spiral_distribution(1000, 0.05)
```

### 🌈 分布的「指紋」

每個分布都有其獨特的「指紋」：

- **螺旋分布**：中心密度高，向外螺旋展開
- **高斯分布**：中心對稱，向外遞減
- **均勻分布**：每個區域密度相等
